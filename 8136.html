linux终端操作

Author: Aaron Date: June 22, 2018  23:35:33 Category: Linux
  1. ctr + alt + t 打开新的终端窗口
  2. ctr + shift + + 终端窗口字体放大
  3. ctr + - 终端窗口字体缩小
  4. ls : 查看目录下的文件信息
  5. pwd: 查看目录所对应路径
  6. touch: 创建文件
  7. mkdir: 创建文件夹
  8. cd: 切换目录
  9. 1 cd .. 切换到上一级目录
  10. 2 cd 目录名： 切换到指定目录
  11. 3 cd -: 切换到上一次路径
  12. 4 cd ~: 切换到用户的家目录 /home/python
  13. 5 cd .: 切换到当前目录
  14. rmdir: 删除空文件夹，注意点:只能是空文件夹ing

sudo -s 获取最顶级的root权限。输入密码
ls -lh (-lh) 更人性化的查看文档
clear 清空
cd- 返回一次的目录
tree 查看目录谁属于谁的
cal 查看当月日历
cal -y 查看当年日历
date 查看日期
history 查看敲的代码的个数
mv 拷贝文件
rm –help (–help) 查看指令是干嘛的
man 查看说明 f翻页，b上一页 q退出
mkdir ./bbb/aa/cc -p 多层嵌套创建的时候在后面加-p
rm 1.txt -i 删除空文件不能删除文件夹 y是确认 n是否

gedit 1.txt 打开文件进行操作写入
cat 3.txt 查看文件中的信息
more 1.txt 分页查看文档（比较好看的类型） 空格下一页 ，回车一行一行查看 q退出

重定向（>） ：ls /bin > ./4.txt 把bin文件 重定到 4.txt 查看
(>>) : tree 1.txt 4.txt >> 3.txt 追加、合并文件

ls /bin | more 管道（|）查看文档
ln -s 1.txt 1-s.txt 创建软连接，快捷方式
ln 1.txt 1-h.txt 创建硬链接 ，链接全部数据
grep “^d” ./1.txt grep “j$” ./1.txt 搜索文档里面的数据后面加-n显示行数
-i 不区分大小写 -v 取反

gz 是z
tar -zcvf test.tar.gz *.txt 压缩
tar -zxvf ./test.tar.gz 解压
tar -zxvf name.tar.gz -C ./aa 想要解压到指定文件夹时 加-C(大C).

bz2 是j
tar -jcvf name.tar.bz2 *.txt 压缩
tar -jxvf name.tar.bz2 解压
tar -jxvf name.tar.bz2 -C ./

zip：
zip zip.zip *.txt 压缩
unzip zip.zip ./ 解压
unzip -d ./cc zip.zip 解压到指定目录

pwd:使用pwd命令可以显示当前的工作目录，该命令很简单，直接输入pwd即可，后面不带参数。
cd:切换工作目录
ls:查看文件信息
ls是英文单词list的简写，其功能为列出目录的内容，是用户最常用的命令之一，它类似于DOS下的dir命令
Linux文件或者目录名称最长可以有265个字符，“.”代表当前目录，“..”代表上一级目录，以“.”开头的文件为隐藏文件，需要用 -a 参数才能显示。
